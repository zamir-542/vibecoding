# VibeCoding Projects

This repository contains multiple projects developed under the VibeCoding folder. 

## Projects

### 1. [PryrTm Android Application](./android_app/)
An Android application built with Jetpack Compose that displays prayer times from the Jakim E-Solat API and provides notifications.

### 2. [E-Book Sales Platform](./website_payment/)
A web-based e-commerce platform that uses Netlify Webhooks and the Resend API for email delivery.

### 3. [CMS Landing Page](./cms_landingpage/)
A landing page managed by a backend content delivery system.

---

## Commit Guidelines
Since multiple projects share this repository, please use directory-scoped prefixes for your commit messages to keep the history clear:
* `feat(android_app): [message]`
* `fix(cms_landingpage): [message]`
* `docs(website_payment): [message]`
* `chore(root): [message]`
