# VibeCoding Monorepo

Welcome to the **VibeCoding Monorepo**! This mathematically structured root repository houses all the independently developed projects under a single, globally organized GitHub namespace.

## 📁 Repository Architecture

This monolith actively contains three wholly isolated architectures:

### 1. [📱 PryrTm Android Application](./android_app/)
A premium, battery-optimized Android application completely built in Jetpack Compose. It dynamically pulls live Muslim prayer time data natively from the Jakim E-Solat API and effortlessly tracks time via a 0% CPU-drain background Glance Widget engine and strict Dual-Phase AlarmManager notifications.

### 2. [💻 E-Book Sales Platform](./website_payment/)
A gorgeous web-based e-commerce platform cleanly wrapped with automated Netlify Cloud Background Webhooks. It directly interfaces with Resend API for rapid, protected email delivery upon purchase completion.

### 3. [🖥️ Headless CMS Landing Page](./cms_landingpage/)
An interactive marketing landing page securely managed by an independent backend content delivery system. 

---

## 🔒 Monorepo Contribution Standards
Because these directories act as isolated universes within the same global repository, **all commits must strictly adhere to Native Directory-Scoped Formatting** to keep the Git timeline mathematically pure:
* `feat(android_app): [your message]`
* `fix(cms_landingpage): [your message]`
* `docs(website_payment): [your message]`
* `chore(root): [your message]`
